Creating an HTML page

You include a Powersoft report (PSR) on a Web page with the Embed element. Element attributes specify the space allocated for the report and the report file name.

A sample Embed element might look like this:

 <EMBED src=April_sales.psr WIDTH=370 HEIGHT=320>

Attributes of the Embed element

The Embed element is part of the HTML specification for plug-ins. For the DataWindow plug-in, you specify only standard HTML attributes.

HTML attributes HTML attributes name the file to be downloaded to the client and the space reserved for the plug-in on the Web page.

Table 32-3: HTML attributes for the DataWindow plug-in
HTML attribute Value
SRC A URL identifying the object to be downloaded.

When the browser processes the Embed element, it requests the resource from the server and finds the DLL in its Plugins directory that handles the content type.

For a DataWindow plug-in, the object is a PSR file.
WIDTH The width of the viewing window in pixels.
HEIGHT The height of the viewing window in pixels.

Sample page

Here is HTML code that includes the DataWindow plug-in in a sample page. Note the use of the Embed element to specify a PSR file.

Opening element

 <HTML>

Document heading

 <HEAD>
<TITLE>On Leave report</TITLE>
</HEAD>
<BODY>

H1 heading

 <H1>On Leave report</H1>

Embed element in a paragraph

 <P><EMBED src=DWB_att_crossout.psr WIDTH=680 HEIGHT =
    350></P>

Closing elements

 </BODY>
</HTML>
