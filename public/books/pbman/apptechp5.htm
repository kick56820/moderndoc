Opening the sample applications

To open a sample application, you need to create a workspace for the application and then add the target for that application to your workspace.

If there is no target file for an application, you can still add it to your workspace. Select Existing Application from the Target tab of the New dialog box. In the wizard that opens, navigate to the PowerBuilder 10.5.0\Code Examples folder, expand the folder and PBL that contains the application you want to examine, and select the application.
