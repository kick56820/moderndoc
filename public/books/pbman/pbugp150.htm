Saving a pipeline

When you have generated a pipeline definition in the Data Pipeline painter workspace, you should save the pipeline. You can then reuse it later.

To save a pipeline:

  1. Click the Save button, or select File>Save from the menu bar.


For a new pipeline

When you save a pipeline for the first time, you must specify a name. The name can be any valid identifier with up to 40 characters. A common convention is to prefix the name with the string pipe_. You can also specify the library in which the pipeline is saved.
