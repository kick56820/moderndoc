About PowerBuilder HTML Books

PowerBuilder has two kinds of online Help: HTML Help and Windows Help.

About HTML Help

This HTML Help contains several PowerBuilder books: the PowerBuilder User's Guide, Application Techniques, the DataWindow Programmer's Guide, Connecting to Your Database, and, for PowerBuilder Enterprise users, Working with Web and JSP Targets.

Context-sensitive Help and reference information for Web and JSP targets is also in HTML format in a separate HTML Help file that is launched when you select Help in a Web or JSP target. The ability to build Web and JSP targets is an optional feature in the Enterprise edition of PowerBuilder. The context-sensitive and reference Help is installed only if you install the feature.

About Windows Help

The Windows Help contains a list of new features in this release, as well as context-sensitive Help and reference information. For information about new features in this release, see the New Features topic in the Help.

To view the contents of the Windows Help file, select Contents from the Help menu.

The Windows Help contains all the reference information in Objects and Controls, the DataWindow Reference, and the PowerScript Reference, as well as for tools such as the Migration Assistant and the Profiling and DWSyntax tools.

For more information about using online Help in PowerBuilder, see Using online Help.
