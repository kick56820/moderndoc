Chapter 13 Using MobiLink Synchronization

About this chapter

The PowerBuilder MobiLink Synchronization for ASA wizard creates objects that make it easy for you to control database synchronization from a PowerBuilder application. This chapter provides an introduction to MobiLink synchronization. It also describes how to prepare to use the wizard, and how to use the objects created by the wizard.
