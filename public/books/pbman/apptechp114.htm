Chapter 20 PowerBuilder Runtime Automation Server

About this chapter

This chapter describes how to access PowerBuilder class user objects by means of automation. A PowerBuilder object is the OLE server; PowerBuilder or any other OLE-enabled development tool can be the client application, accessing the methods and properties of the PowerBuilder object.
